**Task Manager MVP Project**

**User Authentication:** Secure sign-up, login, and password recovery to keep user data private.

**Task Creation:** A simple interface to add task titles and descriptions quickly.

**Due Dates:** Ability to assign a specific deadline to each task with a calendar picker.

**Task Prioritization:** A "High, Medium, Low" flagging system to help users focus on what matters.

**Status Tracking:** Basic states for each task, such as "To Do," "In Progress," and "Completed."

**Task Deletion/Archiving:** The option to remove tasks or move finished ones out of the main view.

**Mobile-Responsive Design:** Ensuring the interface is usable on both desktop and smartphone browsers.

**Search and Filter:** A simple search bar to find tasks by keyword or filter them by their status.
